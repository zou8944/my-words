- **[Hermes Agent v0.11.0 发布：重写 React TUI 和插件系统](https://x.com/WesRoth/status/2047646749427216385)**（来源：X/Twitter @WesRoth）
  > Nous/Hermes 发布 Hermes Agent v0.11.0，引入重写的 React TUI、仪表板插件、主题系统，并支持 DeepSeek V4 和 GPT-5.5。同时发布了原生 Linux 沙箱后端，用于 Deep Agents。

- **[Google/osv-scanner：Go 语言编写的开源漏洞扫描器](https://github.com/google/osv-scanner)**（来源：GitHub Trending）
  > Google 开源的 Go 语言漏洞扫描器，基于 OSV.dev 数据库，支持 11 种语言生态和 19 种锁文件格式。可扫描源码目录、容器镜像和许可证，提供智能修复建议，减少误报并提升漏洞修复效率。
