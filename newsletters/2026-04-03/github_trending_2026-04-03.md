## GitHub Trending


### 1. [anthropics/claude-code](https://github.com/anthropics/claude-code)
> Claude Code 是一款基于终端的智能编程助手，能理解代码库上下文，通过自然语言指令执行常规任务、解释复杂代码并处理 Git 工作流，显著提升开发效率。
---

### 2. [microsoft/VibeVoice](https://github.com/microsoft/VibeVoice)
> 微软开源的VibeVoice是一系列前沿语音AI模型，包含ASR语音识别与TTS语音合成。其核心创新在于采用7.5Hz超低帧率连续语音分词器，结合LLM与扩散模型，能单次处理长达60分钟的音频进行带说话人分轨的转录，或合成90分钟多说话人对话。适用于长音频转录、实时语音输入及多语言语音生成等场景。
---

### 3. [google-research/timesfm](https://github.com/google-research/timesfm)
> TimesFM是谷歌研发的预训练时间序列基础模型，采用解码器架构，支持长达16K的上下文和连续分位数预测。该模型参数精简至2亿，无需频率标识即可进行高精度预测，适用于金融、气象等多领域时序分析任务。
---

### 4. [luongnv89/claude-howto](https://github.com/luongnv89/claude-howto)
> 这是一个针对Claude Code的视觉化、示例驱动的学习指南。它通过结构化教程、可复用的配置模板和Mermaid图表，系统性地教授从基础命令到高级代理编排的全套功能。适用于希望快速掌握Claude Code并将其应用于自动化代码审查、CI/CD等生产场景的开发者。
---

### 5. [axios/axios](https://github.com/axios/axios)
> Axios是一个基于Promise的HTTP客户端，支持浏览器和Node.js环境。它简化了异步HTTP请求的处理，提供拦截器、请求取消等高级功能，适用于前端和后端开发。
---

### 6. [openai/codex](https://github.com/openai/codex)
> OpenAI推出的轻量级本地终端编码助手，支持npm或Homebrew一键安装。作为ChatGPT计划的一部分，它能在终端直接运行，提供代码生成与辅助功能，适合开发者快速集成至工作流，提升本地开发效率。
